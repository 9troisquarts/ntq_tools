## Installation
Add this line to your application's Gemfile:

```ruby
gem "ntq_tools", git: "https://github.com/9troisquarts/ntq_tools"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install ntq_tools
```
